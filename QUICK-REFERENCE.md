# Quick Reference — Second Brain v3

## Capture Commands
```bash
brain "note"            # Dump to inbox
brain --process "note"  # Dump + AI process
```

## Process Inbox
```bash
cd ~/brain && opencode run "Process all notes in 00-Inbox/ using my wiki method. Log everything."
```

## Folders
| Folder | Use For |
|--------|---------|
| 00-Inbox | Raw notes |
| 01-Projects | Things with deadlines |
| 02-Areas | Ongoing (Skills, Finances) |
| 03-Resources | Reference |
| 04-Archive | Processed notes |
| 05-Wiki | Knowledge base |
| 06-Daily | Daily notes |
| 07-Agent-Logs | AI logs |

## The Anson Protocol
- Auto-updates when AI processes notes about you
